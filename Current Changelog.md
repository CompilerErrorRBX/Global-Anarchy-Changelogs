## ⚖️ Changes

### ❗ `Change` Made it more obvious where the menu button is in the class selection UI
> Button now has color. Additionally both the Menu and Close button can exist at the same time if the player opened the class selection UI

### ❗ `Change` Added a "Spawn" button in to the class selection UI for desktop players
> Added a green spawn button to the class selection UI so it is more obvious that we still need to deploy

### ❗ `Change` Made it more obvious where the configure button is when selecting weapons
> Recolored the button blue for better visibility. Also added an icon.

### ❗ `Change` Vote kick now requires server majority to pass the vote
> Previously if the "yays" outnumbered the "nays" your vote would pass. Now you need server majority (includes non voters).
> In the future you will be punished for kicking too many players. So use your kicks wisely.

### ❗ `Change` Explosions on aircraft scorestreaks should deal more consistent damage now
> Previously your explosions would deal different damage to the helicopter depending on where it hit. This creates inconsistent damage feedback.
> The helicopter should now take two RPG shots to take down.

### ❗ `Change` Removed Tertiary slot (Slot 3). It was unused
> This slot was in the original version of the game, and has been long unused. Removing.

### ❗ `Change` You can no longer vault while ADSing or facing backwards
> To try to make vaulting and mantling more explicit we are limiting how much freedom you have with it.
