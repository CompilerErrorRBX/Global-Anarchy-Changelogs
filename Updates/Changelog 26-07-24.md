## 📢 Features and Improvements

### 🗺️ `Map` Suburbs
- Set in central Florida, USA

### ✅ `Feature` End Game Report UI
- Shows at the start of Intermission.
- Displays information about the last match you played.

### 🔼 `Improvement` Squads now have "stickiness" in casual mode
- When the match starts it groups squads with highest priority.
- If you join late, the game will try to keep you with your squad. This is not guaranteed though, as teams may not be balanced.
- If you leave back to the menu another player can take your spot.

### 🔼 `Improvement` When picking up a gun on Controller reloading is now prioritized
